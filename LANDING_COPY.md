## FAQ

- **Is this really free?** Yes. The audit, recommendations, and shareable report are all free.
- **Do you store my data?** By default, no. Audits run in your browser.
- **Is the audit AI-generated?** No. Deterministic rules engine. We use AI only for the optional executive summary.
- **What tools do you support?** Cursor, GitHub Copilot, Claude, ChatGPT, Anthropic API, OpenAI API, Gemini, Windsurf/v0.
- **How accurate are the savings?** We use list pricing as of the latest verification. Annual deals shift the number.
